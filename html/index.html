Transformation and Registration Toolkit

Version 0.16.0

This is the API documentation for the Transformation and Registration Toolkit.

The toolkit comprises several C++ classes dealing with various aspects of registration, curve and surface fitting as well as transformation estimation.

To get started, please have a look a the Tutorial.
