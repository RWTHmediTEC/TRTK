Detailed Description

template<class ValueType = double, class PointType = Coordinate<double>, class DataType = Coordinate<double>>
class TRTK::Interpolation2D< ValueType, PointType, DataType >

Two-dimensional interpolation for irregularly-spaced data.

Template Parameters:
ValueTypeMust be a floating point type.
PointTypeMust provide operator[] for element access.
DataTypeType of the data associated with each point (e.g. intensity, RGB-value, etc.). DataType must support operator+= as well as the multiplication with a scalar of type ValueType.

This class provides a two-dimensional interpolation for irregularly-spaced data. The data types can be arbitrary and must be specified as templates parameters.

The following interpolation methods are available:

  • Inverse Distance Weighting: see getDataInverseDistanceWeighting() for more details.
  • Inverse Distance Weighting KNN: see getDataInverseDistanceWeightingKNN() for more details.
  • Nearest Neighbor: see getDataNearestNeighbor() for more details.

Here is an example of how to use the class:

 #include <iostream>
 #include <TRTK/Interpolation2D.hpp>

 using namespace TRTK;

 int main()
 {
     try
     {
         // Setup data.

         typedef Interpolation2D<double, Coordinate<double>, double> Interpolation2D;
         typedef Interpolation2D::Point Point;

         std::vector<Point> data_points;
         std::vector<Interpolation2D::data_type> data;

         data_points.push_back(Point(1 ,1));
         data.push_back(1);

         data_points.push_back(Point(2, -1));
         data.push_back(2);

         data_points.push_back(Point(0, 0));
         data.push_back(3);

         data_points.push_back(Point(1, 0));
         data.push_back(4);

         data_points.push_back(Point(-1, 1));
         data.push_back(5);

         // Setup interpolation class.

         Interpolation2D interpolation(data_points, data, 0);

         interpolation.setInterpolationMethod(Interpolation2D::INVERSE_DISTANCE_WEIGHTING);
         interpolation.setNumberOfNearestNeighbors(10);
         interpolation.setRadius(10);

         // Get interpolated values.

         std::cout << interpolation(1.2, 0.1) << std::endl;
     }
     catch (std::exception & error)
     {
         std::cout << error.what() << std::endl;
         return -1;
     }

     return 0;
 }

 #include <iostream>

 #include <TRTK/Interpolation2D.hpp>


 using namespace TRTK;


 class RGB
 {
 public:
     RGB(unsigned short red = 0, unsigned short green = 0, unsigned short blue = 0)
     {
         data[0] = red;
         data[1] = green;
         data[2] = blue;
     }

     RGB & operator+=(const RGB & other)
     {
         this->data[0] += other.data[0];
         this->data[1] += other.data[1];
         this->data[2] += other.data[2];

         return *this;
     }

     unsigned short operator[](size_t n) const
     {
         return data[n];
     };

 protected:
     unsigned short data[3];
 };


 RGB operator*(const RGB & rgb, float factor)
 {
     return RGB(rgb[0] * factor, rgb[1] * factor, rgb[2] * factor);
 }

 {
     try
     {
         // Setup data.

         typedef Interpolation2D<double, Coordinate<double>, RGB> Interpolation2D;

Performs an Inverse Distance Weighting.

Parameters:
[in]xx position
[in]yy position

An interpolated value \( data' \) at postion \( (x, y) \) is computed from its surrounding neighborhood \( \mathcal{N} \) by

\[ data' = \left( \sum_{i \in \mathcal{N}} {|d_i|}^{-p} \right)^{-1} \sum_{i \in \mathcal{N}} {|d_i|}^{-p} data_i \]

where \( d_i \) is the distance between the given point \( (x, y) \) and the location of the i-th data point. The Neighborhood is a circle whose radius can be specified by setRadius(). At most max_number_of_nearest_neighbors are used to compute the interpolant, which can be set by setNumberOfNearestNeighbors(). The exponent \( p \) can be set by setExponent().

Exceptions:
std::range_erroris thrown, if no data is given.
std::runtime_erroris thrown, if the search radius is too small and no nearest neighbor are found.
