00383     | m_21  m_22  t_21 |  *  | source_y | =  | target_y |

00398 
00399     <==>    A * x = b
00400     <==>    A^T * A * x = A^T * b
00401     <==>    inverse(A^T * A) * A^T * A * x = x = inverse(A^T * A) * A^T * b
00402 
00403     Thus:
00404 
00405     x = pseudo_inverse * b = [ inverse(A^T * A) * A^T ] * b
00406 
00407     */
